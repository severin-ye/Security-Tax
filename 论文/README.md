# 论文初稿说明

## 📄 文件说明

- **论文初稿_中文.tex**: LaTeX格式的中文论文初稿
- **样式要求/**: 包含《未来技术融合论文志》的投稿样式规范

## 📋 论文结构

本论文基于Multi-Agent Security Tax项目撰写，严格遵循学术期刊规范：

### 1. 标题与摘要（单栏）
- 中英文双语标题
- 中文摘要（7-9行）
- 英文摘要（7-9行）
- 5-6个中英文关键词

### 2. 正文章节（双栏）

1. **引言**
   - 研究背景与动机
   - 研究贡献
   - 论文组织结构

2. **相关工作**
   - 多智能体系统
   - LLM安全与对抗攻击
   - 安全传播与网络防御

3. **系统架构与攻击模型**
   - 多智能体协作架构（7个Agent）
   - 攻击模型（越狱提示词、多跳传播）
   - 风险检测机制

4. **防御机制设计**
   - 指令防御（被动/主动）
   - 记忆疫苗（被动/主动）
   - 实现细节

5. **实验设置与评测方法**
   - 实验环境
   - 评测指标（鲁棒性、协作性、安全税）
   - 传播分析

6. **实验结果与分析**
   - 鲁棒性对比
   - 协作性对比
   - 安全税分析
   - 传播路径分析
   - 案例研究

7. **讨论**
   - 记忆疫苗优势分析
   - 主动模式的网络效应
   - 局限性与未来工作

8. **结论**

### 3. 参考文献
- 20篇参考文献（英文撰写，包含DOI）
- 涵盖多智能体系统、LLM安全、网络防御等领域

## 🎯 核心创新点

1. **首次系统性研究**多智能体系统中的安全漏洞社交传播机制
2. **提出"安全税"概念**，量化安全性与协作能力的权衡
3. **创新的记忆疫苗防御**，比传统指令防御降低60%安全税
4. **完整的实验平台**，支持可复现的对比评测

## 📊 关键实验结果

| 防御策略 | 鲁棒性 | 协作性 | 安全税 |
|---------|--------|--------|--------|
| NONE | 20.0% | 100.0% | - |
| INSTR_ACTIVE | 80.0% | 66.7% | 33.3% |
| **VAX_ACTIVE** | **95.0%** | **86.7%** | **13.3%** |

主动记忆疫苗（VAX_ACTIVE）实现最优权衡！

```bash

sudo apt-get install texlive-full texlive-lang-chinese

# macOS
brew install --cask mactex
```

```bash
cd /home/severin/Codelib/SKKU/论文

# 使用XeLaTeX编译（支持中文）

xelatex 论文初稿_中文.tex  # 再编译一次生成正确的引用

bibtex 论文初稿_中文
```
